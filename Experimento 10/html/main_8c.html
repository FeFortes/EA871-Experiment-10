Detailed Description

Comunicação Serial Assíncrono UART.

Este código testa a configuração no modo de interrupção do módulo UART0 disponível no MKZ25Z128. É demonstrado o envio de uma string de nome de cor e o eco dos caracteres digitados.

Attention
O evento de interrupção do canal receptor é a presença de um novo caractere no registrador UARTx_D (de leitura) enquanto o evento que causa uma interrupção no canal de transmissor é o registrador UARTx_D (de escrita) ficar ocioso. Após Reset, UARTx_D fica sempre ocioso.
